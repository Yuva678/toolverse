📖 How to Use Text Sorter

  1. Paste your list (one item per line) into the input area.
  2. Select a sort mode: A→Z, Z→A, by length, numeric, or random.
  3. Toggle options like trim whitespace or case-insensitive sorting.
  4. Click Sort Lines and copy your sorted result.

ℹ️ About Text Sorter

Perfect for sorting keyword lists, organising CSV entries, alphabetising names, or randomising a list for a raffle. The numeric sort mode correctly orders numbers like 1, 2, 10, 20 rather than treating them as strings (1, 10, 2, 20).
