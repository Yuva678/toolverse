📖 How to Use Base64 Tool

  1. Select Encode to convert plain text to Base64, or Decode to reverse it.
  2. Paste your input text in the text area.
  3. Click Convert to see the result instantly.
  4. Use Swap to send the output back as input for chaining operations.

ℹ️ What is Base64?

Base64 encodes binary data into ASCII text using 64 printable characters (A–Z, a–z, 0–9, +, /). It's widely used to embed binary data in JSON, XML, email, and HTML. Base64 encoding increases data size by ~33%. It is an encoding scheme, not encryption — encoded data can be decoded by anyone.
