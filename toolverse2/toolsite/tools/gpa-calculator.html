📖 How to Use GPA Calculator

  1. Each row represents one course. Enter a course name (optional).
  2. Select the letter grade you received from the dropdown.
  3. Enter the number of credit hours for that course.
  4. Add more courses with Add Course, then click Calculate GPA.

ℹ️ About GPA Calculator

GPA (Grade Point Average) is a weighted average of your grades, where each course's grade is multiplied by its credit hours. This tool uses the standard US 4.0 scale. A GPA above 3.7 is typically considered excellent, while 3.0–3.7 is good standing.
